Мини-приложение на Python для загрузки тендеров с сайтов b2b-center.ru и rostender.info, сохранения данных в CSV и предоставления API-доступа.
Приложение загружает до 100 тендеров с rostender.info и до 20 тендеров с сайта b2b-center.ru (т.к. на сайте отключена пагинация для неавторизованных пользователей).

Принцип работы: извлекаются ключевые данные по карточкам (ID, наименование, дата публикации, срок окончания, цена, место поставки, заказчик, организатор, ссылка, источник),
некоторые снаружи, некоторые изнутри карточки. Далее результат сохраняется в файл CSV, для удобства для обоих сайтов была создана универсальная таблица.
Также реализовал CLI интерфейс и FastAPI endpoint /tenders для получения JSON.

Пример запуска:

CLI
python main.py --source b2b --max 10 --output tenders.csv #(max 20)
python main.py --source rostender --max 50 --output result.csv
где --source - сайт, с которого берем тендеры, --max - нужно число последних тендеров, --output имя выходного файла

FastAPI

Запускаем APi через консаоль командрой "uvicorn api:app --reload"
Переходим в браузере по адресу http://127.0.0.1:8000/docs, жмем кнопку "try it out", далее заполняем 3 поля, source - (b2b | rostender), max - (число необходимых тендеров), 
save_to (путь на компьютере к месту, куда бы вы хотели сохранить CSV)

GET запрос
Запустить APi через консаоль командрой "uvicorn api:app --reload"
В командной строке прописываем: curl "http://127.0.0.1:8000/tenders?source=b2b&max_tenders=3"
